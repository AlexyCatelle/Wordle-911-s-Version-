# 🔥 Wordle 9-1-1

Un clone du célèbre jeu **Wordle**, entièrement réalisé en **JavaScript**, **HTML** et **CSS**, avec un thème inspiré de la série **9-1-1** 🧯🚨.

---

## ✨ Fonctionnalités

- 🧠 Jeu de type Wordle (6 tentatives pour deviner un mot de 5 lettres)
- ⌨️ Saisie possible au clavier **physique** ou via un **clavier virtuel AZERTY**
- 📱 Interface **responsive** et mobile-friendly
- 📢 Popups animées pour les messages de victoire, défaite ou erreur
- 📸 Affichage d’**images** en fonction du message
- ♻️ Bouton de redémarrage intégré
- 🔤 Dictionnaire de mots customisé basé sur l’univers de la série 9-1-1
